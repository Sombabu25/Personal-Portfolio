# 💼 Sombabu Patel - Personal Portfolio


A modern, responsive, and animated personal portfolio website built with React and Tailwind CSS. Showcasing my projects, skills, and experience as a Computer Science student and Full Stack Developer.

##  Live Demo

**[🌐 View Live Portfolio](https://sombabu25.github.io/Personal-Portfolio/)**

## ✨ Features

- **🎨 Modern Design** - Clean, professional layout with dark theme
- **📱 Fully Responsive** - Optimized for all devices (desktop, tablet, mobile)
- **⚡ Smooth Animations** - Engaging animations using CSS and Framer Motion
- **🎯 Interactive UI** - Hover effects, smooth scrolling, and dynamic navigation
- **📧 Contact Integration** - Direct email and social media links
- **📄 Resume Download** - Easy resume download functionality

## 🛠️ Tech Stack

- **Frontend Framework:** React.js
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animation:** CSS Animations & Transitions
- **Deployment:** GitHub Pages
- **Version Control:** Git & GitHub
